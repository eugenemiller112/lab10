The feature of the text that I am measuring is whether a code chunk was included in a question or answer. That is, whether the string "<code>" is in a question or an answer.

The proportion of questions that invoke the "<code>" string is 0.5953 and the proportion for answers is 0.601425.

The mean score for a question without code is 21.8369162, however if the poster includes code in their question it is 27.7450025. Likewise, for a answer with no code the mean score is 4.2338958 and if code is included 18.5056325. Therefore both good questions and good answers include code. This makes sense, as Stackoverflow is a forum based on coding questions. Communicating directly via code is more relevant to the theme of the board, and this is why posts involving code have a higher average score.
